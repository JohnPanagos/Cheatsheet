CSS Table Reference

Framing

Name Description
content Usually the middle of where the text is located.
Padding An element's padding is the space between its content and its border.
Border Border properties allow you to specify the style, width, and color of an element's border.
Margin sets the margins for an element, and is a shorthand property for the following properties.
Display specifies the display behavior (the type of rendering box) of an element.
Background Specifies an image to use as the background of an element.

Colors

Attribute Description
Background-Color You can set the background color for HTML elements.
Color You can set the color of text.
Border You can set the color of borders.
